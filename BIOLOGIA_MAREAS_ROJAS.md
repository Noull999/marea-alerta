## 🌊 ¿QUÉ ES UNA MAREA ROJA?

Una marea roja es una **proliferación explosiva de fitoplancton** (algas microscópicas) que tie­nen dos características:

1. **Abundancia extrema**: Millones a billones de células en metros cúbicos de agua
2. **Toxicidad**: Muchas especies producen toxinas que matan peces y envenenan mariscos

### Nombre Científico
- **HAB** (Harmful Algal Bloom) = Floración de Algas Dañinas
- **Dinoflagelado** = Tipo de célula móvil con flagelo
- **Diatomea** = Tipo de célula con caparazón de sílice

## 🧬 MECANISMO BIOLÓGICO - POR QUÉ OCURREN

### Paso 1: Condiciones Base Normales
En océano abierto normalmente:
- **Pocas células**: 1,000 - 100,000 células/mL (fondo marino)
- **Equilibrio**: Depredadores (zooplancton) comen algas
- **Estabilidad**: Nutrientes limitados = crecimiento lento

### Paso 2: Evento Disparador (Upwelling)
```
Viento poleward (hacia el polo)
        ↓
Agua fría del fondo sube a la superficie
        ↓
NUTRIENTES abundantes (NO₃⁻, PO₄³⁻, Si)
        ↓
"Todas las especies de fitoplancton reciben fertilizante gratis"
```

**Duración**: 2-4 semanas

### Paso 3: Crecimiento Exponencial
```
Condiciones ideales = BLOOM

Factores necesarios:
✓ Nutrientes (entregados por upwelling)
✓ Luz (agua clara, no muy profundo)
✓ Temperatura óptima para especie
✓ Agua estable (poca mezcla)
✓ Tiempo (2-4 semanas)

Resultado:
Células: 100,000 → 1,000,000 → 10,000,000 células/mL
(Clorofila: 0.3 → 1.0 → 3.0+ mg/m³)
```

### Paso 4: Selectividad de Especies
```
Aquí es donde entra LA TEMPERATURA

En upwelling normal: Muchas especies crecen
- Diatomeas (Pseudo-nitzschia) si T < 14°C
- Dinoflagelados (Gymnodinium) si T > 14°C
- Haptofitas si hay suficiente silicio

PERO en AGUA ANÓMALAMENTE CÁLIDA:
- Gymnodinium catenatum DOMINA
- Crece más rápido que competidores
- Sobrevive mejor que otras especies
- Produce más toxina

← AQUÍ ES TU INSIGHT CORRECTO
```

### Paso 5: Producción de Toxinas
```
Gymnodinium catenatum produce: PSP (Paralytic Shellfish Poison)
├─ Mecanismo: Bloquea canales de sodio en nervios
├─ Síntesis: Ocurre en fase estacionaria (cuando nutrientes se agotan)
├─ Cantidad: 100-1000 fg/célula (femtogramos)
└─ Efecto: 1,000,000 células/mL = 100-1000 µg/kg en agua

Umbral de peligro:
- < 400 µg/kg = SEGURO (sin closure)
- 400-1000 µg/kg = ALERTA (posible closure)
- > 1000 µg/kg = CUARENTENA (cierre obligatorio)

Acumulación en mariscos:
- Mejillones absorben toxinas del agua
- No se intoxican pero acumulan toxina
- Humanos comen mejillones → intoxicación → parálisis
```

### Paso 6: Colapso del Bloom
```
El bloom se colapsa por:

A) Agotamiento de nutrientes
   - Todas las células comieron disponibles
   - Falta N y P → crecimiento cero
   - Las células se mueren

B) Mezcla vertical del agua
   - Olas aumentan
   - Células dispersadas a > profundidad
   - Menos luz → menos fotosíntesis

C) Depredación
   - Zooplancton (copépodos) comen dinoflagelados
   - Pero Gymnodinium tiene toxina = matan depredadores
   - Es mecanismo de defensa

D) Cambio de temperatura
   - Agua se enfría
   - Especies pierde ventaja competitiva
```

## 🌡️ EL PAPEL CRÍTICO DE LA TEMPERATURA

### Por Qué Gymnodinium Domina en Agua Cálida

**Rango óptimo para Gymnodinium catenatum**:
- **Mínimo**: 10°C (crece muy lentamente)
- **Óptimo**: 12-16°C (máximo crecimiento)
- **Máximo**: 18°C (sobrevive pero lentamente)

**Competidores**:
- **Pseudo-nitzschia**: 8-14°C (mejor en agua fría)
- **Chattonella**: 15-25°C (mejor en agua tropical)
- **Diatomas genéricas**: 6-12°C (agua fría/templada)

### Mecanismo a Nivel Molecular

```
Temperatura afecta:

1. TASA DE CRECIMIENTO (µ = growth rate)
   - Gymnodinium: µ = 0.2 divisiones/día a 12°C
   - Gymnodinium: µ = 0.4 divisiones/día a 16°C
   - Pseudo-nitzschia: µ = 0.1 divisiones/día a 16°C
   
   → A 16°C, Gymnodinium crece 4x más rápido
   → En 2 semanas: 1 célula → 10⁸ células (Gymnodinium)
                → 1 célula → 10⁴ células (Pseudo-nitzschia)

2. TOLERANCIA A ESTRÉS
   - Alta T = estrés en agua fría
   - Baja T = estrés en agua cálida
   - Gymnodinium tolerante a T > 14°C
   - Competitors tolerantes a T < 14°C

3. SÍNTESIS DE TOXINA
   - Gymnodinium produce más toxina a T > 14°C
   - Es mecanismo de competencia
   - "Si soy cálido-tolerante, produzco toxina para defender territorio"

4. COMPOSICIÓN DE MEMBRANA
   - Lípidos de membrana cambian con T
   - Enzimas funcionan a diferentes rangos
   - Gymnodinium optimizado para T > 12°C
```

## 📊 CICLO COMPLETO DE MAREA ROJA

### Timeline Típico en Chiloé

```
Día 0-5: UPWELLING EVENT
├─ Viento poleward fuerte
├─ Agua fría+nutrientes sube
├─ Fitoplancton comienza a crecer
└─ Clorofila: 0.3 → 0.8 mg/m³

Día 6-15: CRECIMIENTO EXPONENCIAL
├─ Gymnodinium domina (si T > 14°C)
├─ Competidores mueren (prefieren T < 14°C)
├─ Población: 1,000,000 → 10,000,000 células/mL
├─ Clorofila: 0.8 → 2.5+ mg/m³
└─ AGUA ROJA VISIBLE

Día 16-25: PRODUCCIÓN DE TOXINA
├─ Células forman esporas reproductivas
├─ Síntesis de toxina aumenta
├─ Agua: 100-500 µg/kg toxina
├─ Mejillones se envenenan
└─ IFOP declara ALERTA

Día 26-35: DECLINACIÓN
├─ Nutrientes agotados
├─ Olas aumentan (mezcla)
├─ Células mueren
├─ Clorofila decrece
└─ Evento termina

Total: 30-35 días desde upwelling hasta closure
```

### Variantes Según Condiciones

**Si hay upwelling débil**:
- Menos nutrientes = crecimiento más lento
- 40-60 días hasta visible
- Puede no llegar a closure

**Si hay agua muy tranquila**:
- Células se concentran en superficie
- Visibilidad roja más rápida
- Toxina se acumula 2x más rápido

**Si temperatura muy alta (> 18°C)**:
- Gymnodinium sufre estrés por calor
- Otras especies pueden dominar
- Posible Chattonella en lugar de Gymnodinium

## 🧪 REQUISITOS ESPECÍFICOS PARA CADA ESPECIE

### Gymnodinium catenatum (Chiloé)

```
REQUISITOS:
├─ Temperatura: 12-16°C (óptimo 14°C)
├─ Salinidad: 32-35 ppt (agua marina normal)
├─ Luz: > 1% irradiancia superficial
├─ Nutrientes: NO₃ > 2 µmol/L, PO₄ > 0.1 µmol/L
├─ Estratificación: Agua estable (MLD < 30m)
└─ pH: 7.8-8.2 (normal marino)

TOXINA:
├─ Tipo: PSP (saxitoxin + derivados)
├─ Síntesis: Cuando N agotado (estrés)
├─ Cantidad: 100-500 fg/célula
└─ Efecto: Bloquea Na+ channels → parálisis

RANGO GEOGRÁFICO EN CHILE:
├─ Norte: Puerto Montt (ocasional)
├─ Centro: Chiloé (frecuente)
├─ Sur: Punta Arenas (raro)
└─ No ocurre: Norte de Puerto Montt (T > 18°C)
```

### Pseudo-nitzschia (Diatomea)

```
REQUISITOS:
├─ Temperatura: 8-14°C (prefiere frío)
├─ Silicio: Si(OH)₄ > 2 µmol/L (necesita para caparazón)
├─ Nutrientes: NO₃ > 5 µmol/L (necesita más que Gymnodinium)
├─ Luz: Similar a Gymnodinium
└─ pH: 7.8-8.2

TOXINA:
├─ Tipo: DA (Domoic Acid)
├─ Raro en Chiloé (prefiere agua más fría)
├─ Común en California, España, Japón
└─ Más letal que PSP (muerte vs. parálisis)

RANGO EN CHILE:
├─ Muy raro en Chiloé
├─ Aparece en eventos de T muy baja
├─ Más común sur de Taitao (Aysén)
```

### Chattonella (Raphidophyte)

```
REQUISITOS:
├─ Temperatura: 15-25°C (agua CÁLIDA)
├─ Luz: Requiere fotoperíodo largo
├─ Nutrientes: Moderado
└─ Especial: Prefers baja salinidad (estuarios)

EFECTO:
├─ No produce toxina en humanos
├─ Produce toxina para PECES
├─ Causa mortandades masivas de peces
└─ Evento devastador para acuicultura

OCURRENCIA EN CHILE:
├─ Rara en Chiloé
├─ Más común en agua tropical/subtropical
├─ Podría aumentar con cambio climático (T↑)
```

## 🌡️ ANOMALÍA DE TEMPERATURA = CLAVE

### Por Qué +1.5°C es Crítico

```
Escenario A: Agua normal de Chiloé
├─ Temperatura base: 11°C (austral spring)
├─ Con upwelling: 10°C (agua fría del fondo)
├─ Resultado: Mix de especies, sin Gymnodinium dominancia
└─ Clorofila aumenta pero sin toxina PSP extrema

Escenario B: Agua anómalamente cálida
├─ Temperatura base: 13°C (anomalía +2°C)
├─ Con upwelling: 12°C (sigue siendo cálido)
├─ Resultado: Gymnodinium DOMINA completamente
├─ Crecimiento 3-4x más rápido
├─ Toxina máxima
└─ CLOSURE GARANTIZADA

← ESTO ES LO QUE PASÓ EN 2016 (Chiloé)
Anomalía SST +2-3°C durante meses
Resultado: CRISIS (50 mil toneladas de salmón muerto)
```

### Anomalía = Desviación del Clima Local

```
Baseline normal (30 años promedio):
├─ Primavera (Sept-Nov): 11°C
├─ Verano (Dic-Feb): 14°C
├─ Otoño (Mar-May): 13°C
├─ Invierno (Jun-Ago): 10°C

Anomalía POSITIVA (+1.5°C):
├─ Primavera: 12.5°C ← Gymnodinium optimal
├─ Resultado: Especie dominante cambia
└─ Riesgo: 10x más alto

Anomalía NEGATIVA (-1.0°C):
├─ Primavera: 10°C ← Pseudo-nitzschia optimal
├─ Resultado: Especies frías dominan
└─ Riesgo: Bajo (aunque clorofila alta)
```

## 🔄 CADENA DE TRANSFERENCIA DE TOXINA

```
Agua marina (100 células/mL Gymnodinium)
        ↓ [1 µg/kg toxina]
        ↓
Mejillón (filtra 50L/día)
        ↓ [acumula sin síntomas]
        ↓ [después de 3-5 días]
        ↓ [1000 µg/kg toxina]
        ↓
Humano come mejillón
        ↓ [30 minutos]
        ↓
SÍNTOMAS: Hormigueo boca, manos, pies
        ↓ [2-4 horas]
        ↓
Parálisis progresiva → muerte por paro respiratorio

Dosis letal: 1800-2000 µg/kg en mariscos
```

## 📈 FACTORES QUE ACELERAN/DESACELERAN

### ACELERAN Bloom

| Factor | Efecto |
|---|---|
| **T anomalía > +2°C** | 4x crecimiento |
| **Nutrientes altos** | 2x crecimiento |
| **Agua muy tranquila** | 2x concentración |
| **Luz máxima (medio día)** | Fotosíntesis optimal |
| **Salinidad normal** | Óptimo |

### DESACELERAN Bloom

| Factor | Efecto |
|---|---|
| **T baja o negativa anomalía** | Crecimiento lento |
| **Nutrientes agotados** | Crecimiento cero |
| **Olas fuertes** | Dispersión, menos luz |
| **Profundidad > 30m** | Menos luz |
| **pH muy bajo** | Estrés celular |

```
Especie dominante: Dinophysis, Alexandrium

Patrón: VERANO (Jun-Sep)

```
Especie dominante: Lingulodinium polyedrum

## 🎯 CONCLUSIÓN: MECANISMO SIMPLE

**Tu comprensión es correcta**:

```
1. UPWELLING → Nutrientes + Agua fría
                ↓
2. ANOMALÍA CÁLIDA → Selecciona Gymnodinium
                      ↓
3. CONDICIONES ESTABLES → Concentración de células
                           ↓
4. TIEMPO (2-4 SEMANAS) → Crecimiento exponencial
                           ↓
5. AGOTAMIENTO NUTRIENTES → Síntesis de toxina
                             ↓
6. ACUMULACIÓN EN MARISCOS → Closure IFOP
```

**Lo que NO importa**:
- Olas (dispersan, pero NO causan)
- Viento (causa upwelling, luego no importa)
- Precipitación (no correlacionado)

**Lo que IMPORTA**:
- Temperatura (selecciona especie)
- Nutrientes (combustible)
- Estabilidad (concentración)
- Tiempo (crecimiento exponencial)
